# web-scrape-client

Simple frontend using Vue and Quasar 

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

To serve build:
```
npm install serve && npm run serve
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
